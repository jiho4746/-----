인터넷기초
---
### 📍 개요
정보화 사회의 기본 도구인 인터넷을 이해하고 웹 페이지를 개발할 수 있는 능력을 배양하기 위해, 웹 프로그래밍의 기본 기술인 HTML, CSS, 자바스크립트를 배우고, 실습을 통해 웹 애플리케이션을 개발하기 위한 핵심 기술을 익힌다.
### 📍 목표
1. 대표적인 웹 개발 기술인 HTML5, CSS, 자바스크립트를 배우고 웹 페이지 개발에 적용하는 능력을 익힌다.
